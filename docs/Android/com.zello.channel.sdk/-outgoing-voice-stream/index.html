OutgoingVoiceStream

abstract class OutgoingVoiceStream : VoiceStream

Stream sending voice audio to the Zello Channels server. Create an outgoing voice stream by calling startVoiceMessage and stop it by calling stop. After stopping an outgoing voice stream, the object is defunct. To send more voice audio to the server, you will need to create a new stream.
