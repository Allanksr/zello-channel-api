onConnectSucceeded

abstract fun onConnectSucceeded(session: Session): Unit

Called when the session finishes connecting to the server and channel. When onConnectSucceeded is called, the connection to the server is fully established. You can now call startVoiceMessage to start speaking to the channel, and will receive incoming messages when other users speak to the channel.

onConnectSucceeded is called after an automatic reconnect, so be aware that your listener may see onConnectSucceeded called multiple times without onDisconnected being called. See onSessionWillReconnect for more about automatic reconnection.
