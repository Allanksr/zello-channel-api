onIncomingVoiceProgress

abstract fun onIncomingVoiceProgress(session: Session, stream: IncomingVoiceStream, positionMs: Int): Unit

This method is called periodically while receiving audio. This method is called frequently, so avoid doing heaving processing work in it. positionMs is in media time from the beginning of the stream, not in wall time.

Parameters

session - the session containing the stream that is reporting progress

stream - the stream that is reporting its progress

positionMs - time in milliseconds since the stream started. This may not match wall time, especially if the stream has a custom voice receiver that is not passing audio through to the device speaker.
