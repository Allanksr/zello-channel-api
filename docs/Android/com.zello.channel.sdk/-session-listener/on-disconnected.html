onDisconnected

abstract fun onDisconnected(session: Session): Unit

Called when the session finishes disconnecting from the server. If the session is disconnected due to a network change or other unexpected event, this method is not called, and onSessionWillReconnect is called instead. In that case, the session automatically attempts to reconnect. You can prevent automatic reconnect attempts by implementing onSessionWillReconnect and returning false.
