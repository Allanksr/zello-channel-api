onConnectSucceeded

 abstract fun onConnectSucceeded(session: Session): Unit

Called when the session finishes connecting to the server and channel. When onConnectSucceeded is called, the connection to the server is fully established. You can now call startVoiceMessage to start speaking to the channel, and will receive incoming messages when other users speak to the channel.

onDisconnected

 abstract fun onDisconnected(session: Session): Unit

Called when the session finishes disconnecting from the server. If the session is disconnected due to a network change or other unexpected event, this method is not called, and onSessionWillReconnect is called instead. In that case, the session automatically attempts to reconnect. You can prevent automatic reconnect attempts by implementing onSessionWillReconnect and returning false.

This method is called periodically while receiving audio. This method is called frequently, so avoid doing heaving processing work in it. positionMs is in media time from the beginning of the stream, not in wall time.
