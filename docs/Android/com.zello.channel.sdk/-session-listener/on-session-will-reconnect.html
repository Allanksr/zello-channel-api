onSessionWillReconnect

open fun onSessionWillReconnect(session: Session, reason: ReconnectReason): Boolean

Called when the session has become unexpectedly disconnected. If the SessionListener returns true, the session will attempt to reconnect to the Zello Channels server.

Parameters

session - the session that has disconnected

reason - the reason the session was disconnected

Return
true to allow the session to attempt to reconnect to the Zello Channels server, and false to stay disconnected
