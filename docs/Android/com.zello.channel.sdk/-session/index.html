Session

class Session

Main SDK object that manages the outgoing and incoming messages and notifies the client about it's state changes through a callback object.

Types

Builder

 class Builder

Builder objects are used to create instances of Session class.

 val channel: String

the name of the channel to connect to

password

 val password: String?

the account password for username to send to the server during logon. If no password is provided, the session will connect in listen-only mode.

requestTimeoutSec

 var requestTimeoutSec: Long

Requests to the Zello channels server will fail with a timed out error if requestTimeoutSec seconds pass without a response from the server. Defaults to 30 seconds.

sessionListener

 var sessionListener: SessionListener?

A callback that is used to notify the client app about state changes. Must be accessed from the main UI thread.

state

 var state: SessionState

The current state of the session object

username

 val username: String?

the username to send to the server during logon. If no username is provided, the session will connect in listen-only mode.
