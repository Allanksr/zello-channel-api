setLogger

fun setLogger(logger: SessionLogger): Builder

Configures the logging behavior of the Zello channels session. Log messages will be written to the ADB log by default. You can disable logging from the SDK by setting the logger to SessionLogger.NONE or perform custom logging by providing your own implementation of SessionLogger.

Parameters

logger - the SessionLogger implementation to use for this session
